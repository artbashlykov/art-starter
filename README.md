# ART Starter

Плагин WordPress, который позволяет быстро настроить техническую часть сайта, настроить главную страницу (в формате удобной визитки на 7 шаблонов) и настроить страницу 404.

**Версия:** 1.0.7  
**Требования:** WordPress 6.0+, PHP 7.4+

**Официальный репозиторий:** [https://github.com/artbashlykov/art-starter](https://github.com/artbashlykov/art-starter) (публичный)

**Материалы автора:** [https://forge.artbashlykov.ru](https://forge.artbashlykov.ru)

## Возможности

- Мастер настроек: название, фавикон, технические опции и удаление демо-контента
- Технические настройки: постоянные ссылки, HTTPS, комментарии, регистрация
- Главная-визитка на 7 шаблонов: профиль, CTA, ссылки, рекомендация, соцсети
- Кастомная страница 404 с изображением, текстом и кнопками
- Изоляция фронтенд-страниц от стилей активной темы

## Установка из репозитория

1. Склонируйте репозиторий в `wp-content/plugins/art-starter`.
2. Активируйте плагин в админке WordPress.
3. Откройте **Настройки → ART Starter** в админке и настройте главную страницу или 404.

## Обновления (GitHub Releases)

Плагин использует [Plugin Update Checker](https://github.com/YahnisElsts/plugin-update-checker). Репозиторий **публичный** — токен GitHub на сайте не нужен.

Для приватного форка в `wp-config.php` можно задать:

```php
define( 'ART_STARTER_GITHUB_TOKEN', 'your-github-token' );
```
